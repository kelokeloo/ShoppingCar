# 简单的购物车案例
## 功能
- 获取商品列表并显示
- 点击商品会跳转到商品详细信息页面
- 在详细商品页中，可以将商品加入购物车
- 展示购物车

## 技术栈
vue3 vue-router vuex vite axios

## 后台
express
